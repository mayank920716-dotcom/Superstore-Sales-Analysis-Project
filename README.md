# Superstore Sales Analysis using SQL

## Project Overview
This project performs data analysis on retail sales data using SQL. The goal is to extract meaningful insights such as sales trends, top-performing regions, and profitable categories.

---

## Tools Used
- SQL (MySQL)
- Data Analysis Techniques
- Window Functions

---

## Dataset
The dataset contains:
- Order ID
- Order Date
- Ship Mode
- Region
- Category
- Sales
- Profit

---

## Business Questions Solved
- What is the total sales of the store?
- Which region generates the highest revenue?
- Which category is the most profitable?
- What are the top-performing orders?
- How do sales change over time?

---

## Key Insights
- Certain regions contribute more significantly to overall revenue.
- Technology category shows high profitability.
- Sales show monthly trends indicating seasonal demand.
- High-value transactions contribute a major portion of total sales.

---

## How to Run the Project
1. Create table using `queries.sql`
2. Import dataset.csv using MySQL Workbench
3. Run all queries step by step
